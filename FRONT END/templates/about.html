About!

Our web application, "Attack Prediction for Wireless Sensor Networks," leverages advanced machine learning to enhance the security of Wireless Sensor Networks (WSNs). Utilizing Particle Swarm Optimization (PSO) and an Explainable Ensemble model based on XGBoost, this project accurately predicts potential attacks such as Blackhole, Flooding, Grayhole, and TDMA, ensuring robust network protection. The integration of SHAP (SHapley Additive exPlanations) provides transparent insights into model predictions, empowering users to understand the factors driving attack detection. Designed for ease of use, the application allows users to input sensor data and receive real-time predictions, supported by visual explanations. Built with Flask, MySQL, and a user-friendly interface, it ensures secure user authentication and seamless data handling. This project aims to safeguard WSNs in critical applications like environmental monitoring and industrial automation by proactively identifying threats. Developed with a focus on reliability and scalability, it combines cutting-edge AI with practical deployment, making it a vital tool for network administrators and security professionals.
